# Site de Casamento

Site estático simples para o seu casamento.

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Personalize textos, datas, locais e contatos.
3. Se quiser publicar, copie os arquivos para seu host estático ou use GitHub Pages.

## Arquivos principais

- `index.html`: estrutura do site
- `styles.css`: estilo visual responsivo
- `script.js`: função básica do formulário RSVP
